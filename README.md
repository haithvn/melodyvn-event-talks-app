# MelodyVN Event Talks App

A simple tech event schedule application built with Node.js, Express, and vanilla JavaScript.

## Description

This application displays a schedule for a 1-day tech event. Users can view the list of talks, including details like time, title, speaker, and description.

## Features

- **View Schedule:** Displays a clear timeline of events.
- **REST API:** Backend API to serve talk data from a JSON source.
- **Responsive Design:** Simple frontend to present the schedule.

## Technologies Used

- **Backend:** Node.js, Express.js
- **Frontend:** HTML, CSS, JavaScript (Vanilla)
- **Data:** JSON storage

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/haithvn/melodyvn-event-talks-app.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd melodyvn-event-talks-app
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

## Usage

1.  Start the server:
    ```bash
    npm start
    ```
2.  Open your browser and visit:
    ```
    http://localhost:3000
    ```
